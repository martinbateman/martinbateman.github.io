    Introduction

    In this lab you are going to be using the various types of storage. You will export home directories to your client machines within your Windows Domain.

    Instructions

    Firstly following the instructions here share a file system to all your clients. Use the disk that you added last week. You should limit the quota of each user to 1GB.

    Using Active Directory alter the home directory of your users to use this new share. Once this is working test that the quota are working correctly.

    Installing FreeNAS
    Download FreeNAS from either here or Blackboard. You do not need to register your email address in order to down it. FreeNAS can act as both Network Attached Storage and as a simple SAN. Create a new virtual machine (based on FreeBSD 64bit) with 2 CPUs, 2 cores each. 8GB of memory and two 20GB hard disks. You will install the FreeNAS OS on one harddrive and use the second as iSCSI storage.
  1. Insert the downloaded ISO into the virtual machine and boot it
    2. [help]
  2. Select Install/Upgrade
    3. [help]
  3. Select to install to da0. Da0 is just the name of the disk that is automatically assigned by BSD.
    4. [help]
  4. The installer will warn you that the disk is about to be erased, select Yes.
    5. [help]
  5. Select a password to use for the root user. I suggest you use Pa55word.
    6. [help]
  6. The installation will take a few minutes. Once it has finished select OK to go back to the main menu.
    7. [help]
  7. Select option 3 to reboot the system.
    8. [help]
  8. After the system has rebooted you should see the menu below.
    9. [help]
  9. Select option 1 and fill out the networking setting as shown below. This will put the FreeNAS machine on your network and you should be able to ping it.
    10. [help]
  10. Once you have finish this configuration change to your Windows Server. You will need to start a web browser and set its address to http://10.0.1.3/
    11. [help]
