    Intoduction

    In this lab you are going to start to create a user import script. You are going to create the data import part of the script. You will read a CSV file which contains user details. You will check that those details make sense (i.e. that they are valid) and it will produce a username for the user based on thier name.

    Instructions

    In your Windows 10 Virtual Machine download and install Python 3.7 Installer from here.

    The editor for Python is called IDLE. Once Python has finished installing using the search tool to start IDLE. You can use it to write and run Python programs. Since Python is a scripting language there is no need to compile your program before running it. Start IDLE. First it will show you the Python shell. Select File then New. You will see a blank window that looks a little like notepad. Type the progam as shown below

    print ("Hello World!")

    Press F5 to save and run your program. You will be prompted to save your program. Use hello as the filename. Look in the Python shell for the output. If you have any problem ask the lab demonstrator for help.

    Now that you have run your first Python you should create a python script that reads in userimport.csv using the CSV reader shown in lectures. You should ignore the first line as it contains the name of the colums.

    The rules for creating the user names are
    • Take the first letter of the first name and the entire second name and concatanate them. Convert the username to lowercase, e.g.
      • Martin Bateman → mbateman
    • You should ignore any user's where either the first name or surname is missing

    Examples of the expected user input and output are shown in the table below
    FirstnameSurname
    MartinBateman
    LesleyMay
    Mitchell
    Nicky

    The example output is shown below
    mbateman
    lmay

    Advanced

    Attempt to automatically generate a random password for each user. You will have to look up from the Python documentation how to generate random characters and how to concatenate (join) them to a string.
