Introduction

This lab introduces the use of Windows Server and Active Directory. You are going to configure Windows Server and Active Directory to create a domain. Once you have completed this lab you should be able to log into your Windows Server 2016 machine using the newly created accounts.

    Using your Windows Server 2016 you are going to install Active Directory and create a number of users, each user will have different abilities.

    Adding Users

  17. Once the computer has rebooted launch the Server Manager.
    18. [help]
  18. Click on Tools → Active Directory Administrative Center
    19. [help]
  19. Click the domain (co2516) you created on the left hand side
    20. [help]
  20. Double click on Users and on the right navigation pane select New User.
    21. [help]
  21. Fill in your own details for your new user and click OK.
    Make sure you include the Administrator group for your new user. This will mean your new user can log into the Windows Server 2016 machine.
    22. [help]
  22. Log out and attempt to log in with your new account.
    23. [help]
