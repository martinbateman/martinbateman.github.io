    Introduction

    In this lab you will configure Windows Server to use SNMP.

    Instructions
  1. On Windows Server start the Server Manager
    2.  
  2. Select "Role-based or feature-based installation" and click next
    3.  
  3. Select "Select a server from the server pool" and the Server Pool area. Select your server and click Next.
    4.  
  4. Click next on the "Select server roles" page.
    5.  
  5. In the "Features" page select "SNMP Service"
    6.  
  6. In the "Features" page select "Remove Server Administration Tools" $\rightarrow$ "Feature Administration Tools" $\rightarrow$ "SNMP Tool"
    7.  
  7. Select Next
    8.  
    Configuring SNMP
  8. Bring up the Services control (you can just search for it)
    9.  
  9. Go to SNMP Services
    10.  
  10. Right click and select Properties
    11.  
  11. Select the Security panel
    12.  
  12. Under Accepted Security Names click Add
    13.  
  13. Leave the community rights as READ ONLY

    14.  
  14. Set the Community Name to public
    15.  
  15. Using the examples given in the lecture as a starting point query use snmpget and snmpwalk to query your newly installed SNMP service.
    16. [help]
    You will need to download and install snmpget and snmpwalk.
