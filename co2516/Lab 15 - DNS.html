    Introduction

    In this lab you will be exploring the DNS system and writing a Python script to interact with it.

    Introduction
    Setting up the DNS
  1. On your Windows Server. Start the Server Manager.
    2.  
  2. Select tools then DNS.
    3.  
  3. Double click on forward look up zones.
    4.  
  4. Double click on co2516.ceps.uclan.ac.uk
    5.  
  5. In the right hand panel, right click and select New Host (A or AAAA).
    6.  
  6. Enter client as the hostname and 10.0.1.10 as the IP address.
    7.  
  7. Select create associated pointer (PTR) record.
    8.  
  8. Click add host.
    9.  
  9. Check that your record has been entered correctly by using the nslookup command from the command prompt.
    10.  
    Python
  10. On your Windows Server. Download and install Activestate Python 3.8.
    [help]
    You may need to switch your machine from host only networking to NAT in order to download packages from the Internet. Remember to switch it back again afterwards.
  11. Click the bottom left of the screen and type in pypm. Select PyPM.
    12.  
  12. In the command prompt that appear type pypm install dnspython.
    13.  
  13. Look up what pypm is and make sure you understand what it is doing.
    14.
