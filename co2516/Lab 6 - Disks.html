In this lab you are doing to investigate disk usage on VMWare and Windows. You will be adding a new disk to your Windows Server.
    Instructions
  1. Turn off your Windows Server.
    2. [help]
  2. Select VM → Settings
    3. [help]
  3. Click Add
    4. [help]
  4. Select Harddrive and click Next
    5. [help]
  5. Select Create new disk and click Next
    6. [help]
  6. Select SCSI and click Next
    7. [help]
  7. Leave the disk size as 20 GB and click Next
    8. [help]
  8. Change the file name. Replace with -0 with -1 and click Finish
    9. [help]
    Software configuration
  9. Start Windows Server
    10. [help]
  10. Start the Server Manager and select Dashboard
    11. [help]
  11. Select Files and Storage
    12. [help]
  12. Select Disks. You should see two disk. The first stores c:\, the second is offline and is our new disk
    13. [help]
  13. Right click on the new disk and select Bring Online.
    14. [help]
  14. Click Yes.
    15. [help]
  15. Click 'To create a new volume start the new volume wizard'
    16. [help]
  16. Click Next
    17. [help]
  17. Click Ok
    18. [help]
  18. Click Next
    19. [help]
  19. Click Next
    20. [help]
  20. Change the Volume label to Home
    21. [help]
  21. Click Next
    22. [help]
  22. Click Create
    23. [help]
    Advanced
  23. Share this disk with your Windows Client.
    24. [help]
