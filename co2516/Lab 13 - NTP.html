    Introduction

    In this lab you will be configuring NTP to create a consistent view of time across your network.

    Instructions
  1. Log in to your Windows Server 2016.
    2. class="collapsible" align="right"> 
  2. Start a command prompt.
    3.  
  3. Stop the Windows time service
    C:\> net stop w32time
    4.  
  4. Configure the NTP server list
    C:\> w32tm /config /syncfromflags:manual /manualpeerlist:"ntp1.uclan.ac.uk, ntp2.uclan.ac.uk"
    5.  
  5. Tell the domain controller that the sources are trusted
    C:\> w32tm /config /reliable:yes
    6.  
  6. Start the time service
    C:\> net start w32time
    7.  
  7. You can review the time service configuration.
    C:\> w32tm /query /configuration
    8.  
    Advanced
