    Introduction

    In this lab you are going to be creating a graphical user interface. You should use ActiveState Python on a Windows client machine.

    You should create a user interface that allows you to install programs from the network.

    You should look at How to Install Software Available on the UCLan Network for inspiration on how the UI should look. I've included a copy on Blackboard.

    Instructions
  1. You should draw the UI first (this could be on paper or on compuer) and then decide how that design will fit into the grid layout and how you can use the components you have been taught in the lecture.
