    Introduction

    In this lab you are going to configure your router as a simple round robin load balancer. You will then access your web sites using your router's address and it will cluster the two www servers to appear as a single server.

    Instructions
  1. On your router install ipvasm using apt-get.
    2.  
  2. Start the load balancer as follows

    mb@router:~$ sudo ipvsadm -A -t 10.0.0.1:80 -s rr
    mb@router:~$ sudo ipvsadm -a -t 10.0.0.1:80 -r 10.0.1.10 -m
    mb@router:~$ sudo ipvsadm -a -t 10.0.0.1:80 -r 10.0.2.10 -m

    3.  
  3. Test that the load balancer has started. Using your client, start a web browser and access the website at http://10.0.0.1/. Debug if there are any problems.
    4.  
  4. Make the load balancer settings persistent.
    5.  
  5. Reboot your router and check that the load balancing still works.
    6.  
  6. How would you test that this load balancing works correctly. Add these tests to your OneNote.
